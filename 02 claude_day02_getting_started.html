Chapter 1

클로드란? 핵심 개념 5분 정리

사용하기 전에 꼭 알아야 할 개념들을 빠르게 훑어봅니다.

클로드(Claude)는 어떤 AI인가?

클로드는 미국 AI 안전 기업 Anthropic(앤트로픽)이 만든 대화형 AI 어시스턴트입니다. 단순한 챗봇이 아니라 문서 요약, 코딩, 분석, 창작 등 다양한 지식 작업을 도와주는 범용 AI 어시스턴트입니다.

가장 큰 특징은 Constitutional AI(헌법적 AI) 기술을 적용해 안전하고 윤리적으로 동작하도록 훈련되었다는 점입니다. 클로드는 단순히 규칙을 따르는 것이 아니라 원칙에 따라 스스로 판단합니다.

💡 TIP — 어떤 모델을 써야 할까?
처음 시작할 때는 Sonnet 4.6으로 충분합니다. 무료 플랜에서도 Sonnet 4.6을 쓸 수 있으며, 일반 글쓰기·요약·질문 응답 등 대부분의 작업을 훌륭하게 처리합니다. 복잡한 코딩이나 긴 문서 분석이 잦아진다면 Opus 4.6으로 업그레이드를 고려하세요.
✅ 적용방법 — 클로드에 대해 직접 물어보기

상황: 클로드를 처음 열었습니다. 무엇부터 해야 할지 모릅니다.

Chapter 2

계정 만들기: 가입부터 첫 대화까지

5분이면 충분합니다. 단계별로 따라해 보세요.

2-1. 계정 생성 단계별 안내

브라우저에서 claude.ai 접속
크롬, 사파리, 엣지 등 어느 브라우저든 가능합니다. 주소창에 claude.ai를 입력하고 Enter를 누르세요. 별도의 앱 설치 없이 바로 웹에서 사용 가능합니다.
"Sign up" 또는 "시작하기" 클릭
메인 화면에서 Sign up (무료 시작) 버튼을 클릭합니다. 이미 계정이 있다면 "Log in"을 클릭하세요.
가입 방법 선택
세 가지 방법 중 하나를 선택합니다:
  ① Google 계정으로 계속하기 (가장 빠름, 추천)
  ② 이메일 주소로 가입 — 이메일 입력 → 인증 코드 확인
  ③ Apple ID로 계속하기 (iOS/Mac 사용자)
이메일 인증 (이메일 가입 시)
입력한 이메일로 6자리 인증 코드가 옵니다. 메일함을 열어 코드를 복사한 뒤 입력창에 붙여넣으면 됩니다. 코드 유효시간은 10분입니다.
이름 및 기본 정보 입력
이름(닉네임도 가능)과 생년월일을 입력합니다. 만 18세 이상만 사용 가능합니다.
이용약관 동의 후 완료!
서비스 이용약관과 개인정보 처리방침에 동의하면 바로 사용할 수 있습니다. 가입 즉시 무료 플랜이 시작됩니다.
💡 TIP — 모바일에서도 앱으로 사용 가능
App Store(iOS) 또는 Google Play(Android)에서 "Claude"를 검색해 공식 앱을 설치하면 모바일에서도 동일하게 사용할 수 있습니다. 2026년 3월부터는 모바일 앱에서 인터랙티브 차트·다이어그램 시각화도 지원됩니다.

메인 화면 구조

│ - 대화 목록 │ 도와드릴까요? │

│ - 설정(Settings) │ [ 메시지 입력창 ... ] │

  • New Chat (새 대화) — 사이드바 상단의 펜 아이콘 클릭. 새로운 독립 대화를 시작합니다.
  • 대화 목록 — 이전 대화가 날짜별로 저장됩니다. 클릭하면 이어서 볼 수 있습니다.
  • Projects (프로젝트) — Pro 이상 플랜에서 관련 대화와 파일을 폴더처럼 묶어 관리합니다.
  • Settings (설정) — 왼쪽 하단의 사람 아이콘 → 테마, 언어, 메모리 등을 설정합니다.
  • 메시지 입력창 — 화면 아래쪽 텍스트 박스. 파일 첨부 아이콘(클립)도 여기 있습니다.
✅ 적용방법 — 첫 대화 나누기
가입 후 바로 첫 대화를 시작해 봅시다. 아래 프롬프트를 복사해 입력창에 붙여넣고 Enter 또는 보내기 버튼을 누르세요.
🎭 실습 시나리오
상황: 가입을 마치고 빈 채팅창 앞에 앉았습니다.
목표: 클로드와 첫 대화를 시작하고, 내 용도에 맞는 활용법을 안내받기
복사해서 바로 쓰는 프롬프트 #3 — 맞춤형 활용법 안내 요청
안녕 클로드! 나는 [직업/역할]이야. 주로 [주요 작업 1], [주요 작업 2], [주요 작업 3]에 클로드를 활용하고 싶어. ① 내 작업에 가장 유용하게 클로드를 쓸 수 있는 방법 3가지를 알려줘. ② 각 방법마다 실제로 내가 입력하면 되는 프롬프트 예시도 하나씩 보여줘. ③ 초보자가 자주 하는 실수와 그 해결법도 알려줘.
📝 사용 예시 — 대괄호 부분을 본인 상황으로 바꾸세요

[주요 작업] → "SNS 콘텐츠 작성", "논문 요약", "이메일 초안", "수업 자료 제작"
복사해서 바로 쓰는 프롬프트 #4 — 이메일 초안 작성 실습
다음 상황에 맞는 이메일 초안을 작성해줘. [상황] - 수신: 거래처 담당자 김OO 과장님 - 목적: 다음 주 미팅 일정 조율 요청 - 분위기: 정중하지만 친근한 비즈니스 톤 - 원하는 미팅 날짜: 4월 10일(목) 오후 2시 또는 4월 11일(금) 오전 10시 제목까지 포함해서 작성해줘.

Chapter 4

언어·테마·개인설정 세팅하기

클로드를 내 취향에 맞게 커스터마이즈하는 방법을 배웁니다.

4-1. 설정 화면 여는 방법

왼쪽 하단 프로필 아이콘 클릭
사이드바 최하단에 있는 본인 프로필 이니셜(예: 홍길동이면 "홍" 또는 "H") 또는 사진 아이콘을 클릭합니다.
메뉴에서 "Settings" 선택
팝업 메뉴에서 Settings (설정)를 클릭합니다. 모바일 앱은 오른쪽 상단의 ≡ 메뉴 아이콘 → Settings 순서로 접근합니다.
원하는 설정 탭 선택
Appearance(테마), Account(계정), Privacy(개인정보), Memory(메모리) 등의 탭이 있습니다.

4-2. 다크 모드 / 라이트 모드 변경

Settings → Appearance 탭 클릭
설정 화면에서 "Appearance" 또는 "테마" 섹션을 찾습니다.
Theme(테마) 선택
세 가지 옵션 중 하나를 선택하세요:
  • Light — 밝은 흰색 배경 (기본값)
  • Dark — 어두운 배경 (눈이 편한 야간 작업용)
  • System — 기기 설정(OS)에 자동으로 맞춤
저장 — 즉시 적용됩니다
별도 저장 버튼 없이 선택하는 즉시 적용됩니다. 언제든 다시 바꿀 수 있습니다.

4-3. 응답 언어 설정하기

클로드는 별도 언어 설정 없이도 한국어로 질문하면 한국어로 답변합니다. 그러나 명시적으로 지정하면 더 일관성 있는 응답을 받을 수 있습니다.

방법 1: 프롬프트에 언어 지정 (가장 쉬움)
메시지 끝에 "한국어로 답해줘" 또는 "Always reply in Korean" 를 추가합니다. 해당 대화에만 적용됩니다.
방법 2: User Preferences(사용자 환경설정)에 고정
Settings → Profile → "User Preferences" 항목에 아래 문장을 입력하면 모든 대화에 항상 적용됩니다:

항상 한국어로 답변해줘. 기술 용어는 한국어 번역과 함께 영어 원어도 괄호 안에 표기해줘.
방법 3: Project 시스템 프롬프트 활용 (Pro 이상)
특정 프로젝트 안에서만 적용할 규칙을 Project Instructions(프로젝트 지침)에 적어 넣을 수 있습니다. 업무 프로젝트에는 비즈니스 어체, 개인 프로젝트에는 편한 말투 등으로 달리 설정 가능합니다.

4-4. 메모리(Memory) 설정하기

2026년 3월부터 무료 사용자 포함 전체에게 메모리 기능이 제공됩니다. 클로드가 대화와 대화 사이에 나의 이름, 선호도, 진행 중인 프로젝트를 기억합니다.

Settings → Memory 탭 확인
메모리 ON/OFF 토글이 있습니다. 기본값은 ON입니다.
저장된 메모리 내용 확인 및 편집
클로드가 자동으로 기억한 항목들이 목록으로 보입니다. 각 항목을 클릭해 수정하거나 삭제할 수 있습니다. 불필요한 정보가 기억되어 있다면 직접 삭제하세요.
메모리 전체 초기화 (선택)
"Clear all memories" 버튼으로 저장된 모든 메모리를 삭제할 수 있습니다. 개인정보 보호가 필요하다면 사용 후 주기적으로 정리하세요.

4-5. User Preferences(개인 선호도) 설정

Settings → Profile → User Preferences 섹션에 클로드가 항상 참고할 나의 특성과 선호도를 적어 놓을 수 있습니다. 한 번만 설정하면 모든 새 대화에 자동으로 적용됩니다.

복사해서 바로 쓰는 User Preferences 예시 (설정창에 붙여넣기)
- 항상 한국어로 답해줘. - 나는 [직업]이야. 답변할 때 내 직업 맥락을 고려해줘. - 전문 용어는 쉽게 풀어 설명하고, 영어 원어도 괄호 안에 표기해줘. - 답변은 핵심만 간결하게 해줘. 불필요한 인사말은 빼줘. - 코드 예시가 있으면 항상 주석도 한국어로 달아줘. - 내 이름은 [이름]이야.
✅ 적용방법 — 완벽한 개인화 세팅 만들기
아래 프롬프트로 나만의 User Preferences 문장을 클로드에게 대신 만들어달라고 요청할 수 있습니다.
🎭 실습 시나리오
상황: 매번 새 대화마다 "한국어로 답해줘", "간결하게 써줘" 등을 반복 입력하는 게 불편합니다.
목표: User Preferences에 한 번 설정해두고, 모든 대화에 자동 적용되도록 만들기
복사해서 바로 쓰는 프롬프트 #7 — 나만의 User Preferences 생성
내가 클로드 설정의 "User Preferences"에 넣을 문장을 만들어줘. [나에 대한 정보] - 직업/역할: [예: 중학교 수학 선생님] - 주로 사용하는 언어: 한국어 - 클로드 활용 목적: [예: 수업 자료 제작, 문제 풀이 도움] - 선호하는 답변 스타일: [예: 간결하게, 예시 위주로, 단계별로] - 피하고 싶은 것: [예: 너무 긴 서론, 영어로만 된 답변] 이 정보를 바탕으로, 클로드가 항상 참고할 수 있는 "개인 설정 문장"을 3~5줄로 만들어줘.
복사해서 바로 쓰는 프롬프트 #8 — 지금 설정이 잘 적용되는지 테스트
내 User Preferences 설정이 잘 반영되고 있는지 확인하고 싶어. 지금 네가 알고 있는 나에 대한 정보(이름, 직업, 언어 선호, 스타일 등)를 정리해서 보여줘. 빠진 게 있거나 추가로 알아두면 좋을 정보가 있으면 뭘 더 설정하면 좋을지도 제안해줘.
