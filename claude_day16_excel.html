클로드 × 엑셀 — 어떤 방법으로 쓸 수 있나?

클로드와 엑셀을 연결하는 방법은 2가지입니다. 상황에 맞는 방법을 선택해서 사용하세요.

방법 1: Claude.ai 채팅에서 엑셀 질문하기 (무료, 즉시 가능)

엑셀 파일을 업로드하거나, 데이터를 텍스트로 붙여넣거나, 수식·기능에 대해 질문하는 방법입니다. 별도 설치 없이 claude.ai 웹사이트에서 바로 사용 가능합니다.

❌ 기존 방식
구글 검색 → VLOOKUP 예시 찾기 → 내 데이터에 맞게 수정 → 오류 발생 → 다시 검색...

→ 30분 ~ 1시간 소요
✅ 클로드 방식
"A열에 직원명, B열에 부서코드가 있고, 별도 시트에 부서코드-부서명 테이블이 있어. C열에 부서명을 자동으로 가져오는 수식 만들어줘"

→ 30초에 완성 수식 + 설명

방법 2: Claude for Excel 애드인 (Pro 이상, 파일 안에서 직접 사용)

Claude for Excel 애드인을 사용하면 엑셀 파일 안에서 직접 Claude를 호출할 수 있습니다. 파일 전체 컨텍스트를 클로드가 이미 파악한 상태에서 명령어를 입력합니다.

구분방법 1: 채팅방법 2: 애드인
설치불필요 (claude.ai)엑셀 → 추가 기능 → Claude 설치
플랜무료 포함 전체Pro 이상
파일 접근업로드 필요현재 파일 자동 접근
즉각 적용수식 복사·붙여넣기 필요선택 셀에 바로 적용
적합한 용도수식 학습, 아이디어 탐색실제 파일 작업, 반복 업무
💡 오늘 교재 활용 전략
애드인이 없어도 100% 활용 가능합니다. 방법 1(채팅)으로 수식·분석·차트 프롬프트를 익히고, 애드인이 있다면 방법 2의 심화 기능까지 활용하세요. 이 교재의 모든 프롬프트는 방법 1로도 동작합니다.

Section 1 · 수식 생성 기초

🔢 수식 생성 — 설명만 하면 완성 수식이 나온다

수식 문법을 몰라도 됩니다. "무엇을 하고 싶은지"를 한국어로 설명하면 클로드가 정확한 수식과 사용법을 함께 알려줍니다.

수식 요청의 황금 공식 — 5가지 정보 포함

데이터 위치 (어느 셀·열에 있는지)
A열에 날짜, B열에 매출, C열에 지역... 이런 식으로 열 구조를 알려줍니다.
"A2:A100"처럼 정확한 범위를 주면 더 정확한 수식이 나옵니다.
원하는 결과 (무엇을 계산/표시할지)
"서울 지역 매출 합계", "이름이 중복된 경우 표시", "전월 대비 증감률 계산"
결과를 구체적으로 묘사할수록 더 정확한 수식이 만들어집니다.
조건이 있는지 (특정 조건 필터)
"2026년 1분기만", "완료 상태인 것만", "금액이 100만원 이상인 것"
조건이 있으면 SUMIF, COUNTIF, AVERAGEIF 계열이 필요합니다.
수식이 들어갈 셀 위치
"D2에 입력하고 아래로 복사할 거야", "요약 시트의 B5에"
절대참조($)가 필요한지 결정하는 데 중요합니다.
엑셀 버전 (선택사항)
XLOOKUP은 엑셀 2019 이상에서만 작동합니다. 버전이 낮으면 VLOOKUP 대안을 제시합니다.
"Excel 2016 사용 중"처럼 명시하면 호환 수식을 만들어줍니다.

실전 예시 — 이런 데이터 구조를 클로드에게 보여주면

ABCDE
1 날짜 지역 담당자 매출 목표달성률 ← 수식 필요
2 2026-04-01서울김민준4,500,000 =D2/VLOOKUP(C2,목표!$A:$B,2,0)
3 2026-04-02부산이서연3,200,000 =D3/VLOOKUP(C3,목표!$A:$B,2,0)
4 2026-04-03서울박지호5,100,000 =D4/VLOOKUP(C4,목표!$A:$B,2,0)
🎭 실습 시나리오 1
상황: 직원별 매출 데이터가 있는데, 별도 시트에 있는 직원별 목표값을 가져와서 달성률을 계산해야 합니다. VLOOKUP을 써본 적이 있지만 자꾸 오류가 납니다.
목표: 오류 없이 작동하는 달성률 수식 완성
P-01복사해서 바로 쓰는 프롬프트 — VLOOKUP / XLOOKUP 수식 생성
# 두 시트 간 데이터 참조 수식 자동 생성 엑셀 수식을 만들어줘. [데이터 구조] 시트1 (데이터): - A열: 날짜 - B열: 지역 - C열: 담당자 이름 - D열: 실제 매출 - E열: ← 여기에 달성률 수식이 필요 시트2 (목표): - A열: 담당자 이름 - B열: 월 매출 목표 [원하는 결과] E2에 들어갈 수식: 담당자 이름(C2)을 기준으로 시트2에서 목표를 찾아서 (실제 매출 / 목표) 달성률을 % 형식으로 계산 [조건] - 담당자 이름이 시트2에 없는 경우 "목표 없음" 표시 - E2에 수식을 넣고 아래로 드래그해서 복사할 거야 - 엑셀 버전: [2019 이상 / 2016 이하 중 선택] 수식과 함께: ① 각 부분이 어떤 역할을 하는지 설명 ② E열 전체에 % 형식 적용하는 방법 ③ 자주 나오는 오류 원인과 해결법
P-02복사해서 바로 쓰는 프롬프트 — 수식 오류 진단 & 수정
# 오류 나는 수식을 클로드에게 보내서 원인과 해결책 받기 이 엑셀 수식에서 오류가 나는데 왜 그런지, 어떻게 고치는지 알려줘. [오류 나는 수식] =VLOOKUP(C2,시트2!A:B,2,FALSE) [오류 메시지] #N/A (또는 #VALUE!, #REF! 등 실제 오류 입력) [데이터 상황] - C2에는 "김민준"이라고 입력되어 있음 - 시트2의 A열에도 "김민준"이 있는데 찾지 못함 - [다른 특이사항 있으면 추가] 분석해줄 것: ① 이 오류가 발생하는 3가지 일반적인 원인 ② 내 상황에서 가장 가능성 높은 원인 ③ 각 원인별 해결 수식 ④ 오류가 나도 셀이 비어 보이게 하는 방법 (IFERROR 활용)

복잡한 수식 예시 — 이런 것도 클로드가 만들어줍니다

다중 조건 집계 — 2026년 1분기 서울 매출 합계
=SUMPRODUCT((YEAR(A2:A100)=2026)*(MONTH(A2:A100)<=3)*(B2:B100="서울")*D2:D100)
등급 자동 분류 — 점수 구간별 A/B/C/D 등급
=IFS(C2>=90,"A",C2>=80,"B",C2>=70,"C",C2>=60,"D",TRUE,"F")
동적 배열 — 중복 없는 고유 지역 목록 자동 생성
=SORT(UNIQUE(B2:B1000)) ← 엑셀 365/2021 이상
🎭 실습 시나리오 2
상황: 직원 성과 평가 시트가 있습니다. 점수 구간별 등급, 팀별 평균 계산, 상위 10% 직원 자동 표시가 모두 필요합니다. 각각 수식을 어떻게 써야 할지 모릅니다.
목표: 성과 평가 자동화 수식 세트 완성
P-03복사해서 바로 쓰는 프롬프트 — 복잡한 다중 조건 수식 생성
# 복잡한 비즈니스 로직을 수식으로 자동 변환 직원 성과 평가 시트에 필요한 수식들을 모두 만들어줘. [데이터 구조] A열: 직원 이름 B열: 팀 (영업1팀, 영업2팀, 마케팅팀) C열: 상반기 점수 D열: 하반기 점수 E열: 종합점수 ← 수식 필요 F열: 등급 ← 수식 필요 G열: 팀 내 순위 ← 수식 필요 H열: 전사 상위 10% 여부 ← 수식 필요 [계산 규칙] 종합점수: 상반기 40% + 하반기 60% 가중 평균 등급 기준: 90점 이상=S, 80~89=A, 70~79=B, 60~69=C, 60 미만=D 팀 내 순위: 같은 팀 안에서 종합점수 기준 순위 (동점자 처리 포함) 상위 10%: 전체 직원 중 상위 10%에 해당하면 "★", 아니면 빈칸 [추가 요청] ① 각 수식을 E2, F2, G2, H2에 넣는 버전으로 작성 ② 아래로 드래그 시 자동 적용되는지 확인 ③ 동점자가 있을 때 순위 수식 처리 방법 (RANK.EQ vs RANK.AVG)
P-04복사해서 바로 쓰는 프롬프트 — 날짜 계산 자동화 수식
# 계약 만료, 근속 연수, D-day 등 날짜 관련 수식 모음 아래 날짜 관련 계산들을 엑셀 수식으로 만들어줘. [데이터] A열: 직원 이름 B열: 입사일 C열: 계약 만료일 (있는 경우) [필요한 수식들] ① D열: 근속 기간 (오늘 기준) - "3년 5개월" 형식으로 표시 ② E열: 계약 만료까지 남은 일수 - 음수면 이미 만료된 것 → "만료됨" 표시 ③ F열: 계약 만료 상태 알림 - 30일 이내 만료: "🔴 긴급" - 60일 이내 만료: "🟡 주의" - 그 외: "✅ 정상" ④ G열: 올해 연차 발생일 (입사일 기준 +1년마다) ⑤ H열: 다음 생일까지 남은 일수 - 오늘이 생일이면 "🎂 오늘!" 각 수식에 사용한 함수 간단 설명 포함.
Section 3 · 데이터 분석

📊 데이터 분석 — 피벗테이블·통계·트렌드

엑셀 파일을 업로드하면 클로드가 직접 분석해줍니다. 또는 피벗테이블 설계, 통계 수식, 트렌드 분석 방법을 안내받을 수 있습니다.

방법 A: 파일 직접 업로드해서 분석 요청

claude.ai에서 엑셀 파일을 첨부하면 클로드가 데이터를 읽고 분석합니다. 어떤 인사이트가 있는지, 어떤 차트가 적합한지 자동으로 제안합니다.

방법 B: 데이터를 텍스트로 붙여넣기

엑셀에서 데이터를 복사해서 프롬프트에 붙여넣으면 됩니다. 클로드는 탭 구분 텍스트 형식을 그대로 인식합니다.

엑셀에서 복사한 데이터 → 클로드에 붙여넣기 예시
월 채널 신규고객 매출(만원) 객단가 1월 온라인 234 4,320 18.5 1월 오프라인 156 3,800 24.4 2월 온라인 289 5,120 17.7 2월 오프라인 142 3,650 25.7 3월 온라인 312 5,680 18.2 3월 오프라인 178 4,200 23.6

🤖 Claude for Excel 애드인 — 파일 안에서 직접 사용

Pro 이상 플랜에서는 엑셀 파일 안에 Claude를 직접 불러올 수 있습니다. 현재 파일의 모든 데이터를 파악한 상태로 수식 적용, 분석, 차트 생성이 가능합니다.

애드인 설치 방법

엑셀 → 홈 탭 → 추가 기능(Add-ins) 클릭
리본 메뉴 "삽입" 탭 또는 "홈" 탭에서 "추가 기능" 버튼을 찾습니다. Office 365는 "Get Add-ins" 버튼.
Office 스토어에서 "Claude" 검색 → 설치
스토어에서 "Claude by Anthropic"을 검색해서 설치합니다. 무료 설치이지만 사용에는 Pro 이상 구독이 필요합니다.
Claude 계정 연동
애드인 패널이 열리면 claude.ai 계정으로 로그인합니다. Pro/Max/Team 플랜 사용자는 자동으로 기능이 활성화됩니다.
파일 컨텍스트 공유 → 사용 시작
"Share file context" 버튼을 클릭하면 현재 파일 전체를 Claude가 파악합니다. 이후 패널에서 자연어로 명령하면 바로 적용됩니다.
🔑 애드인의 핵심 강점 — PPT 애드인과 연동
Claude for Excel과 Claude for PowerPoint 애드인을 함께 사용하면, 엑셀에서 분석한 내용이 파워포인트 슬라이드 생성에 자동으로 연동됩니다. 엑셀 분석 → 파워포인트 보고서 생성이 한 흐름으로 가능합니다. (Day 17에서 자세히 다룸)

P-12복사해서 바로 쓰는 프롬프트 — 데이터 정제 & 클렌징 수식 세트
# 지저분한 데이터를 깔끔하게 정리하는 수식 모음 받은 엑셀 데이터가 지저분해. 정제 수식 세트를 만들어줘. [데이터 문제 목록] 해당하는 것 체크: 문제 A — 이름 앞뒤 공백 예: " 김민준 " → "김민준"으로 정리 문제 B — 전화번호 형식 불일치 예: "01012345678", "010-1234-5678", "010.1234.5678" → 통일 문제 C — 날짜 형식 혼재 예: "20260101", "2026/01/01", "Jan 1, 2026" → YYYY-MM-DD 통일 문제 D — 중복 데이터 표시 예: 같은 이름이 여러 줄에 있을 때 첫 번째만 표시, 나머지는 "(중복)" 표시 문제 E — 텍스트 안의 숫자 추출 예: "매출 4500만원" 셀에서 숫자 4500만 추출 문제 F — 빈 셀을 위 셀 값으로 채우기 예: 병합된 셀처럼 비어있는 곳을 위 셀과 같은 값으로 각 문제에 대한 수식과, 가능하면 VBA 매크로 버전도 제공해줘.
P-13복사해서 바로 쓰는 프롬프트 — 수식 설명 받기 (기존 수식 해석)
# 이해 못 하는 복잡한 수식을 클로드가 완전히 해설해줌 이 수식이 뭘 하는 건지 완전히 이해하고 싶어. [분석할 수식] =INDEX($B$2:$B$100,MATCH(1,(A2=$C$2:$C$100)*(D2=$D$2:$D$100),0)) 또는: [여기에 이해하고 싶은 수식 붙여넣기] [설명 요청] ① 이 수식이 전체적으로 하는 일 (한 문장 요약) ② 각 부분을 하나씩 분해해서 설명: - 어떤 함수인지 - 인수가 뭘 의미하는지 - 왜 이 방식을 쓰는지 ③ 이 수식이 필요한 실제 상황 예시 ④ 같은 결과를 내는 더 쉬운 대안 수식 (있다면) ⑤ 이 수식에서 흔히 발생하는 오류와 해결법 비전공자도 이해할 수 있게 쉽게 설명해줘.
P-14복사해서 바로 쓰는 프롬프트 — 마스터: 내 엑셀 파일 완전 자동화 설계
# 내가 매번 수동으로 하는 엑셀 작업을 전부 자동화하는 설계 내가 매달 반복하는 엑셀 작업을 완전히 자동화하는 방법을 설계해줘. [현재 수동으로 하는 작업] 구체적으로 설명해주세요: ① [예: 각 팀에서 엑셀 파일을 받아서 하나로 합치기] ② [예: 합친 데이터에서 월별 통계 계산] ③ [예: 차트 업데이트] ④ [예: 결과물을 이메일로 보내기] [현재 걸리는 시간] [예: 매월 3시간] [설계 요청] ① 각 작업을 자동화할 수 있는 방법 (수식 vs VBA vs 파워쿼리 중 최적) ② 자동화 난이도 (쉬움/보통/어려움) 및 필요 기술 ③ 완전 자동화 시 예상 소요 시간 감소 ④ 지금 당장 시작할 수 있는 "가장 효과 큰 자동화 1가지" ⑤ 단계별 구현 순서 (2~4주 로드맵) Claude for Excel 애드인과 Cowork를 함께 활용하는 방법도 제안해줘.
